### 문제 설명

<p>원 이동하기 2 문제를 만들고 만든 데이터가 문제의 조건에 맞는지 확인하는 코드를 작성해야한다.</p>

<p>해당 문제의 데이터는 아래 조건들을 만족해야한다.</p>

<ol>
	<li>모든 원의 중심 좌표는 $x$축 위에 존재해야 한다.</li>
	<li>$N$개의 원 중 임의의 두 원을 선택했을 때, 교점이 존재하지 않아야 한다. 즉, 하나의 원이 다른 원 안에 존재하거나 외부에 존재한다.</li>
</ol>

<p>데이터 형식은 원의 개수 $N$이랑 각 원의 중심 $x$좌표, 원의 반지름 $r$만 주어진다. 따라서, 2번 조건을 만족하는지만 확인하면 된다.</p>

<p>주어진 데이터가 해당 조건을 만족하는지 확인해보자.</p>

### 입력 

 <p>첫 번째 줄에는 원의 개수 $N$이 주어진다.</p>

<p>두 번째 줄부터 $N+1$번째 줄까지 원의 중심 $x$좌표, 원의 반지름 $r$이 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>데이터가 조건에 맞는다면 <code>YES</code>, 조건에 만족하지 않는다면 <code>NO</code>를 출력한다.</p>
